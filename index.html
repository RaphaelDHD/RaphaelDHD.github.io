Projets

  • 2D map generator

    L'objectif de cette application est de créer des cartes de jeux vidéos en 2D grâce à une intelligence artificielle. L'application génère un fichier tmx afin d'éditer votre carte dans Tiled ! Les graphistes peuvent à présent gagner du temps dans la création de cartes.

    Technologies utilisées : Fooocus, Python, Gradio, Tiled

  • Spizeur

    Projet reprenant le principe d'un marketplace. Réalisé à l'aide de l'API Dummy JSON on peut retrouver toutes les fonctionnalités de base, une connexion et modification de compte, un panier, l'achat de produits et une page de visualisation des produits.

    Technologies utilisées : Kotlin, room, retrofit, mvvm, coroutines

  • Frog survivor

    Premier jeu que j'ai pu réalisé, il reprend le principe du jeu Survivor.io avec un affichage en 3D. On peut retrouver un système de vagues, de statistiques, d'augmentation avec de multiples ennemies.

    Technologies utilisées : Unity, C#

  • Pokemon card shop

    Site web de vente de cartes pokémon. On peut retrouver un système de connexion, de panier, d'achat, de collections, de recherches avec du tri et de la pagination.

    Technologies utilisées : Cake PHP, MySQL

  • Projet RBX

    Projet RBX est une application développée en Flutter. Elle permet de visualiser les monuments présents dans la ville de Roubaix, de les localiser sur une carte et d'en visualiser les détails.

    Technologies utilisées : Flutter, Riverpod, Hive, Connectivity plus
